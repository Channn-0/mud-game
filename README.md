ผลงานสำหรับเข้าประกวดนวัตกรรมเพื่อการพัฒนาประชาธิปไตยที่ยั่งยืน

(ชื่อเกม) เป็นเกมบนเว็บแนว Text-based RPG (MUD - Multi-User Dungeon) ที่ถูกสร้างขึ้นมาเพื่อสื่อสารแก่นแท้ของคำว่า "ประชาธิปไตย" ให้เข้าใจและสื่อสารได้ง่ายมากขึ้น เพื่อปลูกฝังความรู้นี้ให้ยั่งยืนสืบต่อไปปป

บทบาทแรก (Demo Version): ตำรวจ

ในเวอร์ชัน Demo นี้ เราเลือกนำเสนอผ่านบทบาทของ "ตำรวจ" เนื่องจากเป็นอาชีพด่านหน้าของการบังคับใช้กฎหมาย เป็นจุดที่สามารถสะท้อนภาพของประชาธิปไตย จริยธรรม และการทำตามหน้าที่ได้อย่างชัดเจนที่สุด ผู้เล่นจะถูกบีบคั้นให้ต้องตัดสินใจในสถานการณ์ที่กลืนไม่เข้าคายไม่ออก ระหว่าง "คำสั่ง/ระบบ" กับ "ความถูกต้อง/ความเท่าเทียม" ซึ่งทุกการตัดสินใจจะต้องแลกมาด้วยสถานะทางการเงิน หรือแม้กระทั่งความสุขของครอบครัว

🎮 ฟีเจอร์หลัก (Features)

Interactive Storytelling: เนื้อเรื่องที่แตกแขนงไปตามการตัดสินใจของผู้เล่น ทุกคำตอบมีราคาที่ต้องจ่าย

Moral Dilemma & Resource Management: ระบบจัดการทรัพยากรที่ขัดแย้งกันเอง (เงินในบัญชี, ค่าความชอบธรรม/Democracy, ความสุขครอบครัว)

Retro Terminal UI: หน้าจอแสดงผลสไตล์คอมพิวเตอร์ยุคเก่า (CRT Scanline Overlay) เพื่อสร้างบรรยากาศของระบบที่เคร่งครัด

Persistence Mechanic: ระบบ "ซ่อนเงื่อน" ในบางสถานการณ์ที่กดดันให้ผู้เล่นต้องกดปุ่มยืนหยัดต่อสู้กับคำสั่งที่ไม่ชอบธรรมซ้ำๆ

Multiple Endings: ฉากจบหลากหลายรูปแบบ ที่สะท้อนผลลัพธ์จากการกระทำของคุณในสังคมนี้
